### Installation

MYMOID sdk is available on npm [@mymoid/components](https://www.npmjs.com/package/@mymoid/components)

```
npm install @mymoid/components
```

To render the widget inside of you application you must import from `@mymoid/components`

### How use `@mymoid/components`

```
import { PaymentForm } from "@mymoid/components"
```

After that you can inserte `PaymentForm` in you code.

<Playground>
  <PaymentForm brandColor="pink" shortCode="XXXXXX" />
</Playground>

> NOTE: By default mymoid componet will load some style by default.

## Customizing your Widget

To change default styled you can pass `brandColor` property to customize.

<Playground>
  <PaymentForm brandColor="pink" shortCode="XXXXXX" />
</Playground>
